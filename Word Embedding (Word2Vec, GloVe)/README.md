# 👋 Introduction

해당 자료는 LLM에서 전체적인 Work Flow를 이해하기 위해 반드시 필요한 Word Embedding에 대해 공부한 내용입니다.
Word2Vec과 GloVe에 대한 내용이 포함되어 있으며, 각 두 방식을 최대한 자세히 풀어냈습니다.
